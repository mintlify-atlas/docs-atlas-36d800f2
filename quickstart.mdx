This guide will walk you through creating a simple location-based application using Laravel Eloquent Spatial.

## What We'll Build

We'll create a `Place` model that stores locations with points and areas with polygons, then query them using spatial functions.

## Create Your Model

<Steps>
  <Step title="Generate the model and migration">
    Create a new model with a migration file:

    ```bash
    php artisan make:model Place --migration
    ```
  </Step>

  <Step title="Add spatial columns to the migration">
    Open the generated migration file and add spatial columns:

    ```php
    use Illuminate\Database\Migrations\Migration;
    use Illuminate\Database\Schema\Blueprint;
    use Illuminate\Support\Facades\Schema;

    return new class extends Migration
    {
        public function up(): void
        {
            Schema::create('places', function (Blueprint $table) {
                $table->id();
                $table->string('name')->unique();
                $table->geometry('location', subtype: 'point')->nullable();
                $table->geometry('area', subtype: 'polygon')->nullable();
                $table->timestamps();
            });
        }

        public function down(): void
        {
            Schema::dropIfExists('places');
        }
    };
    ```

    <Info>
      The `geometry()` method accepts a `subtype` parameter to specify the geometry type: `point`, `linestring`, `polygon`, `multipoint`, `multilinestring`, `multipolygon`, or `geometrycollection`.
    </Info>
  </Step>

  <Step title="Run the migration">
    Execute the migration to create the table:

    ```bash
    php artisan migrate
    ```
  </Step>

  <Step title="Configure the model">
    Update your `Place` model to use the `HasSpatial` trait and configure the casts:

    ```php
    namespace App\Models;

    use Illuminate\Database\Eloquent\Model;
    use MatanYadaev\EloquentSpatial\Objects\Point;
    use MatanYadaev\EloquentSpatial\Objects\Polygon;
    use MatanYadaev\EloquentSpatial\Traits\HasSpatial;

    /**
     * @property Point $location
     * @property Polygon $area
     */
    class Place extends Model
    {
        use HasSpatial;

        protected $fillable = [
            'name',
            'location',
            'area',
        ];

        protected $casts = [
            'location' => Point::class,
            'area' => Polygon::class,
        ];
    }
    ```

    <Note>
      The `HasSpatial` trait adds all the spatial query scopes to your model.
    </Note>
  </Step>
</Steps>

## Create Spatial Data

Now let's create some places with spatial data.

### Creating Points

```php
use App\Models\Place;
use MatanYadaev\EloquentSpatial\Objects\Point;
use MatanYadaev\EloquentSpatial\Enums\Srid;

// Create a place with a point location
$londonEye = Place::create([
    'name' => 'London Eye',
    'location' => new Point(51.5032973, -0.1217424),
]);

// Create with SRID (WGS84 for GPS coordinates)
$whiteHouse = Place::create([
    'name' => 'White House',
    'location' => new Point(38.8976763, -77.0365298, Srid::WGS84->value),
]);
```

<Info>
  Point coordinates are specified as `latitude, longitude` (Y, X in spatial terms).
</Info>

### Creating Polygons

Polygons are created from one or more LineStrings (rings). The first ring is the outer boundary:

```php
use MatanYadaev\EloquentSpatial\Objects\Polygon;
use MatanYadaev\EloquentSpatial\Objects\LineString;
use MatanYadaev\EloquentSpatial\Objects\Point;

$vaticanCity = Place::create([
    'name' => 'Vatican City',
    'area' => new Polygon([
        new LineString([
            new Point(12.455363273620605, 41.90746728266806),
            new Point(12.450309991836548, 41.906636872349075),
            new Point(12.445632219314575, 41.90197359839437),
            new Point(12.447413206100464, 41.90027269624499),
            new Point(12.457906007766724, 41.90000118654431),
            new Point(12.458517551422117, 41.90281205461268),
            new Point(12.457584142684937, 41.903107507989986),
            new Point(12.457734346389769, 41.905918239316286),
            new Point(12.45572805404663, 41.90637337450963),
            new Point(12.455363273620605, 41.90746728266806), // Close the ring
        ]),
    ]),
]);
```

<Warning>
  Polygon rings must be closed - the first and last points must be identical.
</Warning>

## Access Spatial Data

Retrieve and work with spatial properties:

```php
$place = Place::find($londonEye->id);

// Access Point properties
echo $place->location->latitude;  // 51.5032973
echo $place->location->longitude; // -0.1217424
echo $place->location->srid;      // 0 (or 4326 if specified)

// Convert to different formats
echo $place->location->toJson();  // GeoJSON string
echo $place->location->toWkt();   // Well-Known Text
$array = $place->location->toArray(); // PHP array
```

## Query Spatial Data

The `HasSpatial` trait provides powerful query scopes.

### Find Nearby Places

```php
use MatanYadaev\EloquentSpatial\Objects\Point;

$searchPoint = new Point(51.5, -0.12);

// Find places within 5km (5000 meters)
$nearbyPlaces = Place::query()
    ->whereDistanceSphere('location', $searchPoint, '<', 5000)
    ->get();

// Get places with their distance from the search point
$placesWithDistance = Place::query()
    ->withDistanceSphere('location', $searchPoint)
    ->get();

foreach ($placesWithDistance as $place) {
    echo "{$place->name}: {$place->distance} meters away\n";
}

// Order by distance (nearest first)
$sortedPlaces = Place::query()
    ->orderByDistanceSphere('location', $searchPoint)
    ->get();
```

### Check Spatial Relationships

```php
// Find places within a polygon
$polygon = Polygon::fromJson('{"type":"Polygon","coordinates":[[[51.5,-0.2],[51.6,-0.2],[51.6,-0.1],[51.5,-0.1],[51.5,-0.2]]]}');

$placesInArea = Place::query()
    ->whereWithin('location', $polygon)
    ->get();

// Find areas that contain a specific point
$point = new Point(51.5, -0.12);

$containingAreas = Place::query()
    ->whereContains('area', $point)
    ->get();

// Check if geometries intersect
$intersecting = Place::query()
    ->whereIntersects('area', $otherPolygon)
    ->get();
```

## Working with GeoJSON

GeoJSON is a popular format for geographic data. You can easily convert between GeoJSON and geometry objects:

<CodeGroup>

```php From GeoJSON
use MatanYadaev\EloquentSpatial\Objects\Point;
use MatanYadaev\EloquentSpatial\Objects\Polygon;

// From GeoJSON string
$point = Point::fromJson('{"type":"Point","coordinates":[-0.1217424,51.5032973]}');

// From GeoJSON array
$polygon = Polygon::fromArray([
    'type' => 'Polygon',
    'coordinates' => [
        [
            [-0.2, 51.5],
            [-0.1, 51.5],
            [-0.1, 51.6],
            [-0.2, 51.6],
            [-0.2, 51.5],
        ],
    ],
]);
```

```php To GeoJSON
$place = Place::first();

// To GeoJSON string
$geoJson = $place->location->toJson();
// {"type":"Point","coordinates":[-0.1217424,51.5032973]}

// To GeoJSON array
$array = $place->location->toArray();
// ['type' => 'Point', 'coordinates' => [-0.1217424, 51.5032973]]

// To GeoJSON FeatureCollection
$featureCollection = $place->location->toFeatureCollectionJson();
```

</CodeGroup>

<Note>
  In GeoJSON, coordinates are in `[longitude, latitude]` order (X, Y), which is the opposite of how you create Point objects in PHP.
</Note>

## Complete Example

Here's a complete example putting it all together:

```php
use App\Models\Place;
use MatanYadaev\EloquentSpatial\Objects\Point;
use MatanYadaev\EloquentSpatial\Enums\Srid;

// Create some places
Place::create([
    'name' => 'Big Ben',
    'location' => new Point(51.5007, -0.1246, Srid::WGS84->value),
]);

Place::create([
    'name' => 'Tower Bridge',
    'location' => new Point(51.5055, -0.0754, Srid::WGS84->value),
]);

Place::create([
    'name' => 'Buckingham Palace',
    'location' => new Point(51.5014, -0.1419, Srid::WGS84->value),
]);

// Find places within 1km of a location
$myLocation = new Point(51.5074, -0.1278, Srid::WGS84->value); // Near Trafalgar Square

$nearby = Place::query()
    ->whereDistanceSphere('location', $myLocation, '<=', 1000)
    ->withDistanceSphere('location', $myLocation, 'distance_meters')
    ->orderByDistanceSphere('location', $myLocation)
    ->get();

foreach ($nearby as $place) {
    echo sprintf(
        "%s is %.0f meters away\n",
        $place->name,
        $place->distance_meters
    );
}

// Output:
// Big Ben is 286 meters away
// Buckingham Palace is 980 meters away
```
